## セッションの作成

すべてのインタラクションは `AgiApiClient` から始まります。クライアントは API キーで認証し、サーバー上のターミナルセッションを管理します。

```typescript
import { AgiApiClient } from "@work-with-ai/sdk";

const client = new AgiApiClient(
  "https://api.workwithagi.dev", // またはセルフホスト URL
  "your-api-key",
);

// 新しいターミナルセッションを作成
const session = await client.createSession({
  user_id: "user-123",
  label: "dev-session",
});

console.log("Session ID:", session.session_id);
console.log("WebSocket URL:", session.ws_url);
```

<Aside type="tip">
  API キーはハードコーディングせず、環境変数（`WORKWITHAGI_API_KEY`）に保存してください。
</Aside>

## 完全なサンプル

以下は、クライアント作成からセッション開始、ターミナル表示までを含む Vue シングルファイルコンポーネントの全体例です。

```vue
<script setup lang="ts">
import { ref, onMounted, onUnmounted } from "vue";
import { AgiApiClient } from "@work-with-ai/sdk";
import { AgiTerminal } from "@work-with-ai/vue";

const endpoint = "https://api.workwithagi.dev";
const apiKey = import.meta.env.VITE_AGI_KEY;
const sessionId = ref<string>();
const terminalRef = ref();

const client = new AgiApiClient(endpoint, apiKey);

onMounted(async () => {
  const session = await client.createSession({
    user_id: "user-123",
    label: "quickstart",
  });
  sessionId.value = session.session_id;
});

onUnmounted(async () => {
  if (sessionId.value) {
    await client.destroySession(sessionId.value);
  }
});
</script>

<template>
  <div style="height: 500px">
    <AgiTerminal
      v-if="sessionId"
      ref="terminalRef"
      :endpoint="endpoint"
      :api-key="apiKey"
      :session-id="sessionId"
      theme="dark"
      @connect="() => console.log('Session connected')"
      @output="(data) => console.log('Output:', data)"
    />
    <p v-else>セッションを開始中...</p>
  </div>
</template>
```
