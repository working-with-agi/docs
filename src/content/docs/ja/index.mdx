## 何が手に入るか

<CardGrid stagger>
  <Card title="AI ターミナル" icon="laptop">
    AI アシスタント内蔵の Web ターミナルを、任意の Web アプリに埋め込み。コマンド実行、脅威検知、セッション記録に対応。`<iframe>` または JS SDK で統合。
  </Card>
  <Card title="RAG コード検索" icon="magnifier">
    コードベースをインデックスし、キーワードではなく意図で検索。「認証ミドルウェアはどこ？」と聞けば、ファイル名と行番号が返る。ベクトル埋め込み + Qdrant で動作。
  </Card>
</CardGrid>

## 3行で出荷

```typescript
import { WorkAGI } from "@working-with-agi/sdk";

// AI ターミナルを埋め込み
WorkAGI.terminal("#my-terminal", { theme: "dark" });

// 意図でコード検索
const hits = await WorkAGI.search("決済失敗時のリトライロジック");
// → [{ file: "src/billing/retry.ts", line: 42, score: 0.94, snippet: "..." }]
```

## 誰のためのSDKか？

- **社内開発プラットフォームを構築中？** AI ターミナル + コード検索をゼロから作らず追加。
- **開発者ツールを出荷中？** インテリジェントなコードナビゲーションを製品に組み込み。
- **コーディング教育プラットフォーム？** ブラウザ上で AI アシスト付きターミナルを学生に提供。
