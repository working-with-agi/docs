## 何が手に入るか

<CardGrid stagger>
  <Card title="AI ターミナル" icon="laptop">
    AI アシスタント内蔵の Web ターミナルを、任意の Web アプリに埋め込み。コマンド実行、脅威検知、セッション記録に対応。
  </Card>
  <Card title="RAG コード検索" icon="magnifier">
    コードベースをインデックスし、キーワードではなく意図で検索。「認証ミドルウェアはどこ？」と聞けば、ファイル名と行番号が返る。
  </Card>
  <Card title="React / Vue / Angular" icon="puzzle">
    お使いのフレームワーク向けネイティブコンポーネント。バニラ JS ならコア SDK を直接利用。
  </Card>
</CardGrid>

## フレームワークを選択

<Tabs>
  <TabItem label="React">
  ```tsx
  import { AgiTerminal, useCodeSearch } from "@working-with-agi/react";

  function App() {
    const { search } = useCodeSearch();
    return <AgiTerminal theme="dark" ai onThreat={console.warn} />;
  }
  ```
  </TabItem>
  <TabItem label="Vue">
  ```vue
  <script setup>
  import { AgiTerminal, useCodeSearch } from "@working-with-agi/vue";
  const { search } = useCodeSearch();
  </script>

  <template>
    <AgiTerminal theme="dark" ai @threat="console.warn" />
  </template>
  ```
  </TabItem>
  <TabItem label="Angular">
  ```typescript
  import { AgiTerminalComponent, CodeSearchService } from "@working-with-agi/angular";

  @Component({
    imports: [AgiTerminalComponent],
    template: `<agi-terminal theme="dark" [ai]="true" (threat)="onThreat($event)" />`,
  })
  export class AppComponent {
    constructor(private codeSearch: CodeSearchService) {}
  }
  ```
  </TabItem>
  <TabItem label="Vanilla JS">
  ```typescript
  import { WorkAGI } from "@working-with-agi/sdk";

  WorkAGI.terminal("#my-terminal", { theme: "dark", ai: true });
  const hits = await WorkAGI.search("決済失敗時のリトライロジック");
  ```
  </TabItem>
</Tabs>

## パッケージ

| パッケージ | 説明 |
|---|---|
| `@working-with-agi/sdk` | コア SDK — バニラ JS、フレームワーク非依存 |
| `@working-with-agi/react` | React コンポーネント + hooks |
| `@working-with-agi/vue` | Vue 3 コンポーネント + composables |
| `@working-with-agi/angular` | Angular コンポーネント + services |

## 誰のための SDK か？

- **社内開発プラットフォームを構築中？** AI ターミナル + コード検索をゼロから作らず追加。
- **開発者ツールを出荷中？** インテリジェントなコードナビゲーションを製品に組み込み。
- **コーディング教育プラットフォーム？** ブラウザ上で AI アシスト付きターミナルを学生に提供。
