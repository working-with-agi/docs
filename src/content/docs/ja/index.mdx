## あなたのサービスに、AIの力を

ユーザーがAIに自然言語で指示を出し、AIがデータを読み取り、分析し、結果を返す——そんな体験を、あなたのプロダクトに数行のコードで追加できます。

<CardGrid stagger>
  <Card title="AIと対話して作業する" icon="laptop">
    ユーザーは画面上でAIに話しかけるように指示を出すだけ。「今月のレポートを作って」「問題があれば教えて」——AIがサービス内のデータを理解し、分析からアウトプットの生成まで自律的に進めます。
  </Card>
  <Card title="あなたのデータを理解するAI" icon="magnifier">
    サービスに蓄積されたデータ（ドキュメント、履歴、ナレッジベース）をAIが読み取り可能な形にインデックス化。ユーザーが「先月の売上が落ちた原因は？」と聞けば、関連する情報をAIが見つけて回答します。
  </Card>
  <Card title="複数AIの合議で判断する" icon="star">
    MAGI（Multi Agent General Instrument）モード。3つのAIが賛成・反対・中立の立場からそれぞれ分析し、多数決で結論を出します。一方的な回答ではなく、多角的に検証された判断を提供します。
  </Card>
  <Card title="外部ツールとつながるAI" icon="puzzle">
    カレンダー、プロジェクト管理、チャットツール——ユーザーが普段使っているサービスにAIが直接アクセス。「来週の空き時間に会議を入れて」「チケットを作って」といった操作もAIが代行します。
  </Card>
</CardGrid>

## MAGI — 3人の賢者による合議

3つのAIが同時に起動し、それぞれ異なる視点から分析。投票と合議を経て、**一方的ではない、多角的に検証された結論**を導きます。

[MAGI デモを見る →](/ja/showcase/magi-system/)

## 組み込みは簡単

お使いのフレームワーク向けコンポーネントをインポートするだけ。

<Tabs>
  <TabItem label="React">
  ```tsx
  import { WorkWithAI } from "@working-with-agi/react";

  function App() {
    return <WorkWithAI theme="dark" ai />;
  }
  ```
  </TabItem>
  <TabItem label="Vue">
  ```vue
  <script setup>
  import { WorkWithAI } from "@working-with-agi/vue";
  </script>

  <template>
    <WorkWithAI theme="dark" ai />
  </template>
  ```
  </TabItem>
  <TabItem label="Angular">
  ```typescript
  import { WorkWithAIComponent } from "@working-with-agi/angular";

  @Component({
    imports: [WorkWithAIComponent],
    template: `<work-with-ai theme="dark" [ai]="true" />`,
  })
  export class AppComponent {}
  ```
  </TabItem>
</Tabs>

## どんなサービスに使えるか？

- **SaaS プロダクト** — ユーザーがAIと対話しながら作業できるUIを追加
- **社内ツール** — 社員が自然言語でデータに問いかけるインターフェース
- **カスタマーサポート** — AIがナレッジベースから回答を自律生成
- **データ分析** — 「売上の傾向を教えて」で即座にレポート生成
- **教育プラットフォーム** — 学習者がAIアシスタントと対話しながら課題に取り組む
