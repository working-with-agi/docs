## 課題

プロダクトに AI 搭載の開発者ツールを追加したい — AI アシスタント付きターミナルや、意図を理解するコード検索。ゼロから構築すると数ヶ月かかり、PTY 処理、ベクトルデータベース、埋め込みモデル、リアルタイム WebSocket プロトコルの専門知識が必要です。

## 解決策

WorkWithAGI SDK は、2つのプロダクション対応コンポーネントをドロップインで提供します:

### AI ターミナル

任意のブラウザで動作する Web ベースターミナル。ユーザーが得るもの:

- フル PTY ターミナル（偽シェルではない）
- コンテキストを理解する AI アシスタント内蔵
- コマンドの脅威検知とフィルタリング
- セッション記録と再生
- カスタマイズ可能なテーマとキーバインド

iframe または JavaScript SDK で埋め込み:

```html
<!-- iframe: 設定不要 -->
<iframe src="https://terminal.workwithagi.dev/embed?key=YOUR_KEY"></iframe>
```

```typescript
// SDK: フルコントロール
import { WorkAGI } from "@working-with-agi/sdk";

const term = WorkAGI.terminal("#container", {
  theme: "dark",
  ai: true,
  recording: true,
  onThreat: (alert) => notify(alert),
});
```

### RAG コード検索

任意のコードベースをインデックスし、コードが「何をしているか」で検索:

```typescript
const search = WorkAGI.codeSearch({
  index: "my-project",
  embedModel: "bge-small-en-v1.5",
});

// ディレクトリをインデックス
await search.index("./src");

// 意図で検索
const results = await search.query("API タイムアウトのエラーハンドリング");
// → ファイル名、行番号、関連度スコア、コードスニペット
```

特徴:
- ファイル監視による自動増分インデックス
- OpenVINO アクセラレーテッド埋め込み
- Qdrant ベクトルストレージ（セルフホストまたはマネージド）
- コールグラフトレース（関数の全呼び出し元/先を検索）

## アーキテクチャ

```
あなたのアプリ
  │
  ├── WorkAGI.terminal()  →  WebSocket (Socket.IO) → PTY + AI エージェント
  │
  └── WorkAGI.search()    →  REST API → 埋め込み → Qdrant ベクトル DB
```

両コンポーネントともセルフホスト可能。マネージドサービスを選ばない限り、データはあなたのインフラの外に出ません。
