## AI ターミナル

<Tabs>
  <TabItem label="コア">
    - フル PTY ターミナル（bash, zsh, fish — サーバーで動くものなら何でも）
    - Socket.IO による WebSocket トランスポート
    - モダンブラウザで動作
    - レスポンシブ — コンテナサイズに適応
  </TabItem>
  <TabItem label="AI アシスタント">
    - Claude 搭載のコンテキスト対応 AI
    - 会話に基づくコマンド提案
    - エラーのインライン解説
    - システムプロンプトとガードレールの設定可能
  </TabItem>
  <TabItem label="セキュリティ">
    - コマンドのリアルタイム脅威検知
    - コマンド許可/拒否リストの設定
    - フルリプレイ対応のセッション記録
    - 監査ログのエクスポート（JSON, CSV）
  </TabItem>
  <TabItem label="埋め込み">
    - iframe 埋め込み（設定不要）
    - JavaScript SDK（ライフサイクル完全制御）
    - カスタムテーマ（ダーク、ライト、カスタム CSS）
    - キーバインド設定
    - イベントフック: `onCommand`, `onThreat`, `onConnect`, `onDisconnect`
  </TabItem>
</Tabs>

## KnowledgeHub

<Tabs>
  <TabItem label="取り込み">
    - マルチソース取り込み（コード、ドキュメント、Wiki）
    - ファイル監視による増分更新
    - 20以上のプログラミング言語とドキュメント形式をサポート
    - 除外パターンの設定可能
    - 最適な再現率のためのチャンクサイズ調整
  </TabItem>
  <TabItem label="検索">
    - セマンティック検索 — テキストではなく、必要な情報を記述して検索
    - 結果: ソース、セクション、関連度スコア、スニペット
    - 多言語クエリ対応（英語、日本語など）
    - ソースタイプ、ディレクトリ、プロジェクトでフィルタ
  </TabItem>
  <TabItem label="ナレッジグラフ">
    - ドキュメントとコード間の関係をトレース
    - ソース間の相互参照
    - 深さ設定可能（デフォルト: 3階層）
    - ツール連携用 JSON 出力
  </TabItem>
  <TabItem label="インフラ">
    - ベクトルベースのセマンティックインデックス
    - セルフホストまたはマネージドデプロイ
    - カスタム連携用 REST API
    - Docker Compose でワンコマンドセットアップ
  </TabItem>
</Tabs>

## SDK オプション

| 機能 | iframe | JS SDK |
|---|---|---|
| AI ターミナル | 対応 | 対応 |
| KnowledgeHub | — | 対応 |
| カスタムテーマ | 制限あり | フル |
| イベントフック | — | 対応 |
| プログラム制御 | — | 対応 |
| 設定不要 | 対応 | — |

## 対応環境

- **ブラウザ:** Chrome, Firefox, Safari, Edge（最新2バージョン）
- **サーバー:** Node.js 18+, Python 3.10+
- **デプロイ:** Docker, Kubernetes（Helm チャート）, ベアメタル
- **セルフホスト:** 外部依存なしで動作
