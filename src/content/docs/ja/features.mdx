## コンポーネント機能

<Tabs>
  <TabItem label="AI ターミナル">
    - PTY 対応の Web ベースターミナル
    - Claude によるリアルタイム AI アシスタント
    - 脅威検知とコマンドフィルタリング
    - セッション記録と再生
    - iframe または JavaScript SDK で埋め込み可能
    - WebSocket (Socket.IO) トランスポート
    - カスタマイズ可能なテーマとキーバインド
  </TabItem>
  <TabItem label="エージェントシステム">
    - マルチエージェントオーケストレーション (Claude Code SDK)
    - 標準エージェントロール: Developer, Reviewer, Tester, Architect
    - YAML によるカスタムエージェント定義
    - 進捗ストリーミング付き非同期タスク実行
    - エージェント間デリゲーション
    - 会話履歴とコンテキスト管理
  </TabItem>
  <TabItem label="MCP サーバー">
    - プロセス管理 (Circus, Supervisord)
    - セマンティックコード検索 (GrepAI + Qdrant)
    - ドキュメントアクセス (Context7)
    - カスタム MCP サーバー SDK
    - stdio / SSE トランスポート
  </TabItem>
  <TabItem label="RAG パイプライン">
    - コードベースの自動インデックス
    - ファイル監視による増分更新
    - OpenVINO アクセラレーテッド埋め込み
    - Qdrant ベクトルストレージ
    - テナント分離コレクション
    - コールグラフトレーシング
  </TabItem>
</Tabs>
