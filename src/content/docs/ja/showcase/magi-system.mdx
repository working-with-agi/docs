## WorkWithAGI の MAGI モード

WorkWithAGI SDK では、この合議メカニズムを `magi` モードとして実装しています:

```typescript
import { WorkAGI } from "@work-with-ai/sdk";

const result = await WorkAGI.magi({
  question: "次のプロジェクトで Rust を採用すべきか？",
  context: projectData,
  agents: [
    { name: "MELCHIOR", perspective: "technical_merit" },
    { name: "BALTHASAR", perspective: "team_impact" },
    { name: "CASPER", perspective: "pragmatic_balance" },
  ],
});

console.log(result.verdict);    // "CONDITIONAL APPROVE"
console.log(result.votes);      // [APPROVE, REJECT, CONDITIONAL_APPROVE]
console.log(result.reasoning);  // 統合された最終判断の理由
```

3つのエージェントが同時に起動し、それぞれの視座から独立に分析。投票と合議を経て、多角的に検証された結論を導きます。

*「人は自分が望むものしか見ない。MAGIは望まない視点も見せる。」*
