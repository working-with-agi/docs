## Bring AI to your service

Users give instructions in natural language, and the AI reads their data, analyzes it, and delivers results — add this experience to your product with just a few lines of code.

<CardGrid stagger>
  <Card title="Work with AI, naturally" icon="laptop">
    Users simply talk to the AI on screen. "Create this month's report." "Tell me if there are any problems." — the AI understands the data in your service and autonomously handles everything from analysis to output generation.
  </Card>
  <Card title="AI that understands your data" icon="magnifier">
    Index the data in your service — documents, history, knowledge base — so the AI can read and reason about it. When users ask "Why did sales drop last month?", the AI finds the relevant information and answers.
  </Card>
  <Card title="Multi-AI deliberation" icon="star">
    MAGI (Multi Agent General Instrument) mode. Three AIs analyze from different perspectives — for, against, and neutral — then reach a conclusion by majority vote. Not one-sided answers, but multi-faceted, verified judgments.
  </Card>
  <Card title="AI that connects to your tools" icon="puzzle">
    Calendars, project management, chat tools — the AI directly accesses the services your users already use. "Schedule a meeting next week." "Create a ticket." — the AI handles it.
  </Card>
</CardGrid>

## MAGI — Deliberation by Three Wise Agents

Three AIs launch simultaneously, each analyzing from a different perspective. Through voting and deliberation, they produce **conclusions validated from multiple angles**, not one-sided responses.

## Easy to integrate

Import a component for your framework — that's it.

<Tabs>
  <TabItem label="React">
  ```tsx
  import { WorkWithAI } from "@working-with-agi/react";

  function App() {
    return <WorkWithAI theme="dark" ai />;
  }
  ```
  </TabItem>
  <TabItem label="Vue">
  ```vue
  <script setup>
  import { WorkWithAI } from "@working-with-agi/vue";
  </script>

  <template>
    <WorkWithAI theme="dark" ai />
  </template>
  ```
  </TabItem>
  <TabItem label="Angular">
  ```typescript
  import { WorkWithAIComponent } from "@working-with-agi/angular";

  @Component({
    imports: [WorkWithAIComponent],
    template: `<work-with-ai theme="dark" [ai]="true" />`,
  })
  export class AppComponent {}
  ```
  </TabItem>
</Tabs>
