## MAGI Mode in WorkWithAGI

The WorkWithAGI SDK implements this deliberation mechanism as `magi` mode:

```typescript
import { WorkAGI } from "@working-with-agi/sdk";

const result = await WorkAGI.magi({
  question: "Should we adopt Rust for the next project?",
  context: projectData,
  agents: [
    { name: "MELCHIOR", perspective: "technical_merit" },
    { name: "BALTHASAR", perspective: "team_impact" },
    { name: "CASPER", perspective: "pragmatic_balance" },
  ],
});

console.log(result.verdict);    // "CONDITIONAL_APPROVE"
console.log(result.votes);      // [APPROVE, REJECT, CONDITIONAL_APPROVE]
console.log(result.reasoning);  // Synthesized final reasoning
```

Three agents launch simultaneously, each analyzing independently from their perspective. Through voting and deliberation, they produce conclusions validated from multiple angles.

*"People can only see what they want to see. MAGI shows you what you don't want to see."*
