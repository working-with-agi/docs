## Embedding the AI Terminal

### iframe Integration

The simplest way to embed the AI Terminal:

```html
<iframe
  src="https://terminal.workai.dev/embed?tenant=YOUR_TENANT"
  width="100%"
  height="500"
  allow="clipboard-read; clipboard-write"
></iframe>
```

### SDK Integration

For full control over the terminal lifecycle:

```typescript
import { WorkAGI, AgiApiClient } from "@work-with-ai/sdk";

// REST API client
const api = WorkAGI.api("https://your-server:8600", "your-api-key");
// or
const api = new AgiApiClient("https://your-server:8600", "your-api-key");

// Create session
const session = await api.createSession({ user_id: "user-1", tool: "claude" });

// Mount terminal
const terminal = WorkAGI.terminal({
  container: document.getElementById("terminal"),
  endpoint: "https://your-server:8600",
  apiKey: "your-api-key",
  sessionId: session.session_id,
  theme: "dark",
  fontSize: 14,
  onConnect: () => console.log("Connected"),
  onDisconnect: () => console.log("Disconnected"),
});

terminal.dispose();
```

## Agent System Integration

```typescript
const pane = await api.addPane(session.session_id, {
  tool: "claude",
  role: "reviewer",
});

// Remove when done
await api.removePane(session.session_id, pane.pane_id);
```

## MCP Server Setup

```yaml
# workai-mcp.yaml
servers:
  knowledge-hub:
    transport: sse
    url: http://localhost:8808
  process-manager:
    transport: stdio
    command: workai-mcp-circus
```

## KnowledgeHub Configuration

```typescript
import { KnowledgeClient } from "@work-with-ai/sdk";

const kb = new KnowledgeClient("https://your-hub:8610", "your-api-key");
const results = await kb.search("error handling patterns");
```
