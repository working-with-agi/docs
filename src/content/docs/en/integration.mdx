## Embedding the AI Terminal

### iframe Integration

The simplest way to embed the AI Terminal:

```html
<iframe
  src="https://terminal.workai.dev/embed?tenant=YOUR_TENANT"
  width="100%"
  height="500"
  allow="clipboard-read; clipboard-write"
></iframe>
```

### SDK Integration

For full control over the terminal lifecycle:

```typescript
import { WorkAI } from "@aether-platform/workai-sdk";

const workai = new WorkAI({ apiKey: "...", tenant: "..." });

const terminal = workai.terminal.create({
  container: document.getElementById("terminal"),
  theme: "dark",
  fontSize: 14,
  aiAssistant: true,
  sessionRecording: true,
  onCommand: (cmd) => console.log("Command:", cmd),
  onThreat: (alert) => notifySecurityTeam(alert),
});

// Programmatic interaction
await terminal.execute("ls -la");
terminal.dispose();
```

## Agent System Integration

```typescript
const agent = workai.agents.create({
  role: "reviewer",
  model: "claude-sonnet-4-6",
  tools: ["read_file", "search_code", "run_tests"],
});

const result = await agent.run({
  task: "Review the authentication module for security issues",
  target: "src/auth/",
  output: "markdown",
});

console.log(result.findings);
```

## MCP Server Setup

```yaml
# workai-mcp.yaml
servers:
  knowledge-hub:
    transport: sse
    url: http://localhost:8808
  process-manager:
    transport: stdio
    command: workai-mcp-circus
```

## KnowledgeHub Configuration

```typescript
const hub = workai.knowledgeHub.configure({
  chunkSize: 512,
  watchPaths: ["src/", "docs/"],
  excludePatterns: ["node_modules/", "*.min.js"],
});

await hub.ingest();
const results = await hub.search("error handling patterns");
```
