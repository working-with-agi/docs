## Overview

WorkWithAI is a suite of production-ready components that bring AI-powered capabilities into your development workflow. Instead of building AI integrations from scratch, teams can leverage battle-tested building blocks to ship faster.

## Core Components

### AI Terminal
A web-based terminal with built-in AI assistance, threat detection, and session recording. Embeds directly into your application via iframe or SDK.

### Agent Orchestration
Multi-agent system powered by Claude SDK. Define agents for code review, testing, architecture analysis, and deployment — all working together.

### MCP Integration Layer
Model Context Protocol servers that connect AI agents to your existing tools: process managers, code search, documentation, and more.

### Semantic Code Search
GrepAI-powered code intelligence that understands intent, not just text. Find implementations by describing what they do.

### RAG Pipeline
Vector-backed retrieval augmented generation for your codebase. Automatic indexing, incremental updates, and tenant-isolated search.
