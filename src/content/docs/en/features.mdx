## AI Terminal

<Tabs>
  <TabItem label="Core">
    - Full PTY terminal (bash, zsh, fish — whatever the server runs)
    - WebSocket transport via Socket.IO
    - Works in any modern browser
    - Responsive — adapts to container size
  </TabItem>
  <TabItem label="AI Assistant">
    - Context-aware AI powered by Claude
    - Suggests commands based on conversation
    - Explains errors inline
    - Configurable system prompts and guardrails
  </TabItem>
  <TabItem label="Security">
    - Real-time threat detection on commands
    - Configurable command allowlists/blocklists
    - Session recording with full replay
    - Audit log export (JSON, CSV)
  </TabItem>
  <TabItem label="Embedding">
    - iframe embed (zero config)
    - JavaScript SDK (full lifecycle control)
    - Custom themes (dark, light, custom CSS)
    - Configurable keybindings
    - Event hooks: `onCommand`, `onThreat`, `onConnect`, `onDisconnect`
  </TabItem>
</Tabs>

## KnowledgeHub

<Tabs>
  <TabItem label="Ingestion">
    - Multi-source ingestion (code, documents, wikis)
    - Incremental updates via file watcher
    - Supports 20+ programming languages and document formats
    - Configurable exclude patterns
    - Chunk size tuning for optimal recall
  </TabItem>
  <TabItem label="Search">
    - Semantic search — describe what you need, not exact text
    - Returns: source, section, relevance score, snippet
    - Multi-language queries (English, Japanese, etc.)
    - Filters by source type, directory, or project
  </TabItem>
  <TabItem label="Knowledge Graph">
    - Trace relationships across documents and code
    - Cross-reference between sources
    - Configurable depth (default: 3 levels)
    - JSON output for tool integration
  </TabItem>
  <TabItem label="Infrastructure">
    - Vector-based semantic indexing
    - Self-hosted or managed deployment
    - REST API for custom integrations
    - Docker Compose for one-command setup
  </TabItem>
</Tabs>

## Supported Environments

- **Browser:** Chrome, Firefox, Safari, Edge (latest 2 versions)
- **Server:** Node.js 18+, Python 3.10+
- **Deployment:** Docker, Kubernetes (Helm charts), bare metal
- **Self-hosted:** No external dependencies required
