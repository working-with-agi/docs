## Component Features

<Tabs>
  <TabItem label="AI Terminal">
    - Web-based terminal with PTY support
    - Real-time AI assistance via Claude
    - Threat detection and command filtering
    - Session recording and playback
    - Embeddable via iframe or JavaScript SDK
    - WebSocket (Socket.IO) transport
    - Customizable themes and keybindings
  </TabItem>
  <TabItem label="Agent System">
    - Multi-agent orchestration (Claude Code SDK)
    - Built-in agent roles: Developer, Reviewer, Tester, Architect
    - Custom agent definition via YAML
    - Async task execution with progress streaming
    - Agent-to-agent delegation
    - Conversation history and context management
  </TabItem>
  <TabItem label="MCP Servers">
    - Process management (Circus, Supervisord)
    - Semantic code search (GrepAI + Qdrant)
    - Documentation access (Context7)
    - Custom MCP server SDK
    - stdio and SSE transport
  </TabItem>
  <TabItem label="RAG Pipeline">
    - Automatic codebase indexing
    - Incremental file watching
    - OpenVINO-accelerated embeddings
    - Qdrant vector storage
    - Tenant-isolated collections
    - Call graph tracing
  </TabItem>
</Tabs>
