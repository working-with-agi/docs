## Create a Session

Every interaction starts with an `AgiApiClient`. The client authenticates with your API key and manages terminal sessions on the server.

```typescript
import { AgiApiClient } from "@working-with-agi/sdk";

const client = new AgiApiClient({
  apiKey: "your-api-key",
  endpoint: "https://api.workwithagi.dev", // or your self-hosted URL
});

// Create a new terminal session
const session = await client.sessions.create({
  name: "dev-session",
  env: { NODE_ENV: "development" },
});

console.log("Session ID:", session.id);
```

<Aside type="tip">
  Store your API key in an environment variable (`WORKWITHAGI_API_KEY`) rather than hard-coding it.
</Aside>

## Full Example

Below is a complete Vue single-file component that creates a client, opens a session, and renders a terminal:

```vue
<script setup lang="ts">
import { ref, onMounted, onUnmounted } from "vue";
import { AgiApiClient } from "@working-with-agi/sdk";
import { AgiTerminal } from "@working-with-agi/vue";

const apiKey = import.meta.env.VITE_AGI_KEY;
const sessionId = ref<string>();

const client = new AgiApiClient({
  apiKey,
  endpoint: "https://api.workwithagi.dev",
});

onMounted(async () => {
  const session = await client.sessions.create({ name: "quickstart" });
  sessionId.value = session.id;
});

onUnmounted(async () => {
  if (sessionId.value) {
    await client.sessions.destroy(sessionId.value);
  }
});
</script>

<template>
  <div style="height: 500px">
    <AgiTerminal
      v-if="sessionId"
      :api-key="apiKey"
      :session-id="sessionId"
      theme="dark"
      ai
      recording
    />
    <p v-else>Starting session...</p>
  </div>
</template>
```
