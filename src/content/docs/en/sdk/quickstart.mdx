## Create a Session

Every interaction starts with an `AgiApiClient`. The client authenticates with your API key and manages terminal sessions on the server.

```typescript
import { AgiApiClient } from "@work-with-ai/sdk";

const client = new AgiApiClient(
  "https://api.workwithagi.dev", // or your self-hosted URL
  "your-api-key",
);

// Create a new terminal session
const session = await client.createSession({
  user_id: "user-123",
  label: "dev-session",
});

console.log("Session ID:", session.session_id);
console.log("WebSocket URL:", session.ws_url);
```

<Aside type="tip">
  Store your API key in an environment variable (`WORKWITHAGI_API_KEY`) rather than hard-coding it.
</Aside>

## Full Example

Below is a complete Vue single-file component that creates a client, opens a session, and renders a terminal:

```vue
<script setup lang="ts">
import { ref, onMounted, onUnmounted } from "vue";
import { AgiApiClient } from "@work-with-ai/sdk";
import { AgiTerminal } from "@work-with-ai/vue";

const endpoint = "https://api.workwithagi.dev";
const apiKey = import.meta.env.VITE_AGI_KEY;
const sessionId = ref<string>();
const terminalRef = ref();

const client = new AgiApiClient(endpoint, apiKey);

onMounted(async () => {
  const session = await client.createSession({
    user_id: "user-123",
    label: "quickstart",
  });
  sessionId.value = session.session_id;
});

onUnmounted(async () => {
  if (sessionId.value) {
    await client.destroySession(sessionId.value);
  }
});
</script>

<template>
  <div style="height: 500px">
    <AgiTerminal
      v-if="sessionId"
      ref="terminalRef"
      :endpoint="endpoint"
      :api-key="apiKey"
      :session-id="sessionId"
      theme="dark"
      @connect="() => console.log('Session connected')"
      @output="(data) => console.log('Output:', data)"
    />
    <p v-else>Starting session...</p>
  </div>
</template>
```
