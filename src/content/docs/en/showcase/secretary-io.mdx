## Overview

Secretary.IO leverages the WorkWithAGI SDK to transform meeting recordings into actionable knowledge. Bots join Google Meet / Zoom / Teams sessions to capture audio in real-time, transcription data is indexed via the RAG pipeline, and the agent system summarizes discussions, extracts action items, and identifies decisions.

> "Questions like 'What did who say in last Tuesday's meeting?' are now answered instantly from thousands of hours of meeting data."

## Technology Used

| Component | Purpose |
|---|---|
| **AI Terminal** | Work with AI widget — users interact with AI agent |
| **RAG Pipeline** | Meeting data indexing + semantic search |
| **Agent System** | Minutes summarization, action item extraction, risk analysis |
| **MAGI Mode** | Multi-perspective deliberation for cross-project analysis |
| **MCP Integration** | Bidirectional sync with Linear / Jira / Google Calendar |
